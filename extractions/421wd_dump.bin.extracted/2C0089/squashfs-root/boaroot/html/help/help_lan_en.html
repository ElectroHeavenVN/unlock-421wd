Ipv4 Address

IPv4 Address:Configure IPv4 parameters:

  • IP Address: Input IP address for ONT device and it is not in DHCP Address-asignment pool if DHCP feature is enabled (Default: 192.168.1.1).
  • IP Subnet Mask: An address mask helps us identify the Network ID portion of 32-bit IP address. Usually, it is 255.255.255.0.

DHCP: Device is configured with DHCP server(Dynamic Host Configuration Protocol).

  • Enable/ Disable: This configuration illustrates the way to enable or disable DHCP server. If you disable the Server, you must have another DHCP server within your network or you must configure the IP address of the computer manually
  • IP Address Range: This field specifies the range of address in the IP Address pool. By default, 192.168.1.1 - 192.168.1.254 is the default range of IP address.
  • Lease Time: This is the length of time that network user will be allowed to keep connecting to the ONT device with the current IP Address assigned by DHCP Server. The time range is 1÷3600 (min). The default value is 60 minutes.

DNS:

  • Proxy: This option configures ONT as proxy server helping us resolve domain names
  • Manual: This option help us assign IP Address and there are two way below to implement this
    • Primary DNS: Input the DNS IP address provided by your ISP. Or consult your ISP, default: 8.8.8.8
    • Secondary DNS: You can input the IP Address of another DNS server if your ISP provides two DNS servers, default: 8.8.4.4
  • Wan Interface: It is automatically receive IP from Wan Interface and assigns this IP address to some hosts in LAN network

Static assignment for IP addresses: This option help us assign static IP addresses to some hosts in LAN networks.

  • IP Address: Input IP address and this IP must be in DHCP address pool taht we configure above.
  • MAC Address: Input MAC address of device suppose to have connection with the ONT device.
  • Add: Add new rules in statical IP assigned-address table

IPv6 Address

Router Advertisement configuration(RA):: Configure the Router Advertisement feature in ONT.

  • SLAAC: Enable/ Disable: Enable of diable this feature.
  • Router Advertisement Mode:
    • Automatic: Using parameters retrieving from Wan Interface.
      • Prefix source: Choose WAN Interface to get parameters. By default, WAN interface is interface for Internet service
    • Manual: Allow to change parameters during Router Advertisement process. This configuration below is recommended to be used by advance users:
      • Prefix/Length: Prefix and prefix length to assign IPv6.
      • Preferred Lifetime and Valid Lifetime: the length of time that a valid address is preferred and the length of time that a address is valid.
      • The minimum time and the maximum time between 2 RA packets.

LAN: Configure DHCPv6 server for ONT to allocate IP address for hosts in LAN network.

  • DHCPv6 server: Enable/ Disable: Enable or disable the DHCPv6 servers. If we disable the DHCPv6 server, you must have another DHCPv6 server within your network or you must configure the IPv6 address of the computer manually.
  • Configuration mode:
    • Manual: User configure DHCPv6 mannually. The configuration below is recommended to be used by advance users:
      • IPv6 format: Indetify IPv6 address format or Ipv6 address following EUI64 format.
      • Prefix/Length: Prefix and prefix length to assign IPv6.
      • Prefered Lifetime: the length of time that a valid address is preferred.
      • Valid Lifetime: the length of time that a address is valid.
      • Primary DNS and Secondary DNS: IPv6 address of the DNS servers.
  • Automatic configuration for Prefix and DNS servers: Choose DNS and other paramters configured below
    • DNS Server source: Choose WAN interface to get DNS paramters for IPv6 hosts. By default, WAN interface is the interface using for Internet
  • Automatical configuration for DNS Server: Choose DNS followed by WAN Interface that is configured in "DNS Server Source "

Click APPLY to save and execute the newly set parameters.
