Parental control help parents monitor and manage their children's internet activities, including internet usage time, MAC filtering, and URL filtering.

Parental Control Table: Each row is corresponding to a Parental Control rule (maximum 10 rules)

  • Rule name: Name of the rule
  • MAC Address: The MAC addresses of devices, if multiple MAC addresses are declared, must be separated by comma ",". If the MAC address is FF:FF:FF:FF:FF:FF, the rule will apply to all clients.
  • Access Policy: Allows or denies access for a device. If Access Policy is set to "Scheduled Block based on URL", device access is allowed based on schedules. Access is enabled if there is no schedule defined. If Access Policy is set to "Always Blocks", defined schedules are ignored and no access is allowed for the device.
  • URL: The list of blocked URLs, separated by comma ",". Client can not access to these URLs at any time, leave this field empty if no URLs are blocked.
  • Enable: Enables or disables this rule.
  • Schedules: If Access Policy is set to "Scheduled Block based on URL", clients are permitted to access the internet according to the declared schedules (maximum 4 schedules)
  • Besides, in the beginning of row, there is a checkbox to choose the rules which need to be erased when we click button "DELETE".
